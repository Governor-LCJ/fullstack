- 语义化的url
    :1314/posts GET <!-- GET请求 --> 文章列表 
    :1314/posts/1:id 详情

    POST        新增数据
    PUT         修改数据
    PATCH       修改数据
    DELETE      删除数据

    PUT 完全覆盖数据
    PATCH 部分修改
